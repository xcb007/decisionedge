What you build

Six Excel tools. Yours to keep.

Built during the programme, not after. Real instruments that go straight into your work from day one — not exercises you file away.

01

Decision Framing Dashboard

Structures any problem before analysis begins. Use it before opening any dataset.

02

Hypothesis Tracker

Forces hypothesis-first thinking. Tracks what is confirmed, rejected, and still open.

03

Issue Tree Builder

Breaks complex problems into MECE branches with priority and status tracking.

04

Red Flag Checker

Audits any dataset or AI output for analytical traps. Returns a clear risk rating.

05

Insight Communication Template

A one-pager that leads with the answer — built for a senior audience's 60-second window.

06

Personal Analytics Playbook

Your career positioning, prediction journal, and AI collaboration protocol — reviewed quarterly.
